Decodificador Base64

Como usar:

1. Cole o texto codificado em Base64 no campo superior

2. Clique no botão "Decodificar"

3. O resultado aparecerá no campo inferior

4. Use o botão "Limpar" para reiniciar

Sobre Base64:

Base64 é um método para codificação de dados binários em texto ASCII. É comumente usado para incluir dados em formatos como JSON, XML ou para enviar arquivos via APIs.
